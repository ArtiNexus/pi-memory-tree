# 🧩 Pattern Library

> Reusable patterns discovered by the agent over time.

---

## Pattern Format

Each pattern contains:
- **Category**: workflow | tech | preference | solution | other
- **Title**: Short description
- **Trigger**: When to use this pattern
- **Steps**: What to do
- **Source session**: When first discovered

---
